## A. 절차서에서 파악한 실제 업무흐름 (요약)

기존 PRD가 이미 커버하는 흐름은 생략하고, **보완 요구사항의 근거가 되는 흐름**만 기술한다.

### A.1 정기 위험성평가 연간 사이클 (절차서 8장, 별표1.2)

```
[본사] 전사 실시계획 수립·전 사업장 송부 (5월)
  → [지사] 본사 계획을 반영한 자체 실시계획 수립·보고 (6월)  ※ 필요시 지사 계획 先보고 후 본사 반영
  → 사전준비(안전보건정보·기존 평가자료) → 공정분류 → 유해위험요인 파악(순회점검·근로자 면담·재해/아차사고)
  → 위험성 추정·결정 → 감소대책 수립 → [부장] 확인·승인
  → 결과 보고([안전관리관] → [지사장]) → 근로자·수급업체 공유
  → 감소대책 조치 → 조치 후 허용가능 여부 재확인 → [부장] 확인·승인
  → 환류: 조치결과 반기 1회 이상 취합 → [지사장] 적정성 최종 확인
      → 미조치 사항은 차년도 실시계획에 반영
  → (지사→본사) 이행결과 보고 → 전사 실시결과 보고
```

- 역할 분담(절차서 5장): **안전보건관리책임자(지사장)** 총괄·최종 승인·적정성 최종 확인 / **안전관리관** 계획 수립·결과 작성·취합 보고·산업안전보건위원회 보고 / **관리감독자(부장)** 단계별 확인·승인 / **관리감독자(차·팀장)** 평가 실시·수급업체 평가 검토·결과 공유 / **근로자** 사전준비·전 과정 참여·조치결과 작성·기록 보존.
- 계획에 포함할 항목: 평가 일정, 평가 기법, 평가 인원, **평가 전 교육 실시 방안**, 해당연도 본사 주관부서 실시계획.

### A.2 도급 시 수시 위험성평가 사이클 (절차서 9장, 별표1.3)

```
계약 체결 시 [도급인→수급인] 안전보건정보 제공 (전자계약시스템·공문 등)
   : 사업장 위험성평가 결과, 유해위험물질 정보·MSDS, 작업환경측정 결과, 비상경보체계·대피방법
  → 착수(공) 전 [수급인] 도급인 평가 결과를 반영한 위험성평가 실시표 작성·제출
  → [도급인 차·팀장] '수급업체 위험성평가 이행점검 체크리스트'(별표1.6)로 적정성 점검, [부장] 확인
      → 보완 필요 시 수급인에게 수정 요청 또는 작업 전 수시평가에 반영
  → 작업 전 [도급인+수급인 공동] 수시 위험성평가서 확정
  → 안전작업허가서 발행 시 수시 위험성평가서 첨부, 작업현장 비치
  → 작업 시 [작업자·감독자] 작업단계별 '현장확인' 체크 + TBM 공유 여부 확인
      → 현장에서 추가 위험 발견 시 재평가 후 재비치
  → 반기 1회 이상 '수시 위험성평가 이행점검 체크리스트'(별표1.7) 작성
      → 기실시 수시평가서를 점검하여 **표준모델(작업별 표준 평가 템플릿)에 반영·개정**
```

### A.3 기법 운용 (절차서 6장)

| 실시 유형 | 절차서 지정 기법 |
|---|---|
| 최초·정기 | (일반) KRAS, (작업) JSA, (물질) CHARM |
| 수시 | (작업) JSA, 체크리스트법, (물질) CHARM |
| 공정 | 정성적(HAZOP, K-PSR), 정량적(CA), 화학물질(CHARM), 작업(JSA) 활용 |

- 정해진 기법 외 타 기법 적용 시 **해당 기법 지침을 작성(안전관리관)·승인(안전보건관리책임자)한 후** 실시해야 한다.

## B. 신규 기능 요구사항 (기존 §5에 추가)

### B.1 평가 실시계획 수립·전개 (SCHD 확장)

> 근거: 절차서 8.1(계획 수립), 별표1.2(전사→지사 연간 사이클). 기존 FR-SCHD는 "도래일 알림"까지만 다루고 **계획 문서의 수립·승인·본사↔사업장 전개**가 없다.

| ID | 우선순위 | 요구사항 |
|---|---|---|
| FR-SCHD-010 | P1 | **연간 정기평가 실시계획 수립**: 사업장 단위로 연간 실시계획(평가 일정, 대상 부서/공정, 적용 기법, 평가 인원, **평가 전 교육 실시 방안**, 기타)을 등록하고 결재(승인권자: 사업장 안전관리자 → 안전보건관리책임자)를 받는다. 승인된 계획은 FR-SCHD-002 정기 도래일 관리와 연결된다. |
| FR-SCHD-011 | P2 | **본사 계획 전개**: 본사 총괄이 전사 실시계획(기준 일정·중점 사항)을 등록해 전 사업장에 배포하고, 각 사업장 계획의 수립·승인 현황을 본사 대시보드에서 추적한다(절차서상 5월 전사 계획 → 6월 사업장 계획 순서. 사업장 계획 先수립 후 본사 반영도 허용). |
| FR-SCHD-012 | P1 | **미조치 사항 차년도 이월**: 연말(또는 계획 수립 시점) 기준 미완료 개선조치를 조회해 차년도 실시계획의 '전년도 미조치 반영' 목록으로 일괄 이월·연결할 수 있다(절차서 8.6 환류: 미조치 사항은 차년도 계획 수립 시 반영). |

### B.2 표준모델 관리·이행점검 (ASMT/SCHD 확장)

> 근거: 절차서 8.2.5, 9.6, 별표1.7. 절차서의 "수시 위험성평가 표준모델"은 작업 유형별 표준 평가 템플릿이며, **실사용 평가서를 점검해 표준모델을 개정하는 반기 피드백 루프**가 제도화되어 있다. 기존 FR-ASMT-006(요인 라이브러리)·FR-ASMT-J-004(JSA 템플릿 복제)에 저장·재사용은 있으나 **점검→개정 사이클**이 없다.

| ID | 우선순위 | 요구사항 |
|---|---|---|
| FR-ASMT-014 | P1 | **표준모델(작업 표준 템플릿) 관리**: 작업/공정 유형별 표준 평가 템플릿(요인·대책 세트 포함)을 버전 관리하며 등록·개정·폐기할 수 있다. FR-ASMT-006/J-004의 템플릿을 이 표준모델로 일원화하고, 평가서 생성 시 어느 표준모델(버전)에서 파생됐는지 기록한다. |
| FR-SCHD-013 | P1 | **수시평가 이행점검 체크리스트**: 반기 1회 이상(정기평가 시점 포함), 기간 내 실시된 수시평가서 목록을 불러와 "작업내용·유해위험요인의 표준모델 반영 필요 여부"를 점검하고 결과(해당 표준모델명, 반영 내역: 추가/변경/삭제)를 기록한다. 작성자(차·팀장급)와 확인자(부장급)의 확인 서명이 남고, 반영 필요 항목은 FR-ASMT-014 표준모델 개정 작업과 연결된다. 반기 내 미실시 시 경보를 표시한다. |

### B.3 유해위험요인 파악 활동 기록 (ASMT 확장)

> 근거: 절차서 8.2.2, KRAS장 3절, 별표2.3(파악표 4종). 절차서는 요인 파악의 **방법과 그 기록 자체**(순회점검·청취조사·자료조사·체크리스트 조사표)를 법정 보존 대상으로 요구한다. 기존 PRD는 평가서 행에 요인을 바로 입력할 뿐 "파악 활동"의 기록이 없다.

| ID | 우선순위 | 요구사항 |
|---|---|---|
| FR-ASMT-012 | P1 | **유해위험요인 파악 조사표**: 평가서(주로 KRAS)에 파악 활동 기록을 첨부·연결할 수 있다. 유형 4종 — ① 사업장 순회점검 조사표(수행자·일시·발견 작업/장소·사고/질병 유형) ② 청취조사 조사표(근로자 면담, 육하원칙 서술, 근로자·수행자 의견) ③ 안전보건자료 조사표(재해조사보고서·작업환경측정·건강진단·아차사고 자료 기반) ④ 안전보건 체크리스트 조사표. KRAS 정기평가는 특별한 사정이 없으면 **순회점검에 의한 파악을 포함**해야 한다는 점을 검증 규칙(경고)으로 반영한다. 조사표에서 파악한 요인을 평가서 요인 행으로 가져올 수 있다. |
| FR-ASMT-013 | P2 | **요인 행 재해이력 태그**: 요인 행 비고에 아차사고 / 산업재해 / 동종(업종)사고 태그를 선택 입력할 수 있다(절차서 별표2.4·6.6·7.2 실시표 비고란 공통). 제보(FR-SHRE-005)·수시 트리거 사유 ⑤와 연계해 통계(FR-DASH-004)에 재해이력 기반 필터를 추가한다. |

### B.4 도급인 안전보건정보 제공 (ORG/SHRE 확장)

> 근거: 절차서 9.1. 기존 PRD의 협력사 플로우는 "협력사→자사 제출·승인"의 단방향이다. 절차서는 그 이전 단계로 **도급인→수급인 방향의 안전보건정보 제공**(계약 체결 시)을 의무화한다.

| ID | 우선순위 | 요구사항 |
|---|---|---|
| FR-SHRE-008 | P1 | **도급 안전보건정보 제공**: 협력사–사업장 계약(FR-ORG-006) 등록·갱신 시, 도급인 측이 해당 협력사에 제공할 안전보건정보 패키지를 구성·발행할 수 있다 — ① 해당 사업장·작업 관련 위험성평가 결과(최신 승인본) ② 유해위험물질 정보·MSDS ③ 작업환경측정 결과 ④ 비상시 경보체계·대피방법 ⑤ 기타. 협력사 관리자는 자기 업체에 발행된 정보를 열람·다운로드하고 열람 확인이 기록된다. 제공 이력(일시·항목·수신 업체)은 도급인 의무 이행 증빙으로 보존된다. |

### B.5 수급업체 평가 이행점검 체크리스트 (APRV 확장)

> 근거: 절차서 9.2, 별표1.6. 기존 FR-APRV-004/005는 승인/반려 처리만 규정한다. 절차서는 검토를 **정형 체크리스트**로 남기고 검토자·확인자 2단 서명을 요구한다.

| ID | 우선순위 | 요구사항 |
|---|---|---|
| FR-APRV-009 | P1 | **수급업체 위험성평가 이행점검 체크리스트**: 협력사 제출 평가서의 검토 단계에서 정형 점검표를 작성한다 — ① 위험성 판단기준 보유 여부 ② 위험도(중대성) 반영 내용의 적정성 ③ 판단기준에 따른 감소대책 수립 항목의 적용 여부 ④ 평가 시 해당 작업 근로자의 참여·공유 여부 + 종합 의견(사전 제공 자료, 점검·검토 결과, 보완 조치 필요사항). 검토자·확인자가 각각 서명하며, 점검표는 평가서에 첨부·보존되고 승인/반려 판단의 근거로 출력물에 포함된다. |

### B.6 JSA 현장확인 (ASMT-J 확장)

> 근거: 절차서 9.5, 별표3.2 실시표의 '현장확인' 열. 승인된 JSA가 **작업 당일 현장에서 단계별로 확인**되는 것까지가 절차서의 완결 흐름이다. 기존 PRD는 승인 후 공유·서명에서 끝난다.

| ID | 우선순위 | 요구사항 |
|---|---|---|
| FR-ASMT-J-005 | P1 | **작업 당일 현장확인 체크**: 승인 완료된 JSA에 대해 작업일 단위로 '현장확인 세션'을 생성하고, 작업단계별로 **작업 전 / 작업 중** 각각 작업자(V)·감독자(O)가 조치 이행 여부를 체크한다(모바일 우선). 체크자·일시가 기록되며, 현장 확인 중 추가 위험 발견 시 해당 JSA의 재평가(개정)로 연결하는 버튼을 제공한다(절차서 9.5.3). 세션에는 "TBM 등을 통한 참여 인원 공유 확인" 체크 항목을 포함한다(9.5.5 — TBM 회의 관리 자체는 기존 범위 제외 유지, 확인 체크만 지원). |
| FR-ASMT-J-006 | P2 | **현장 비치용 출력**: JSA 승인본을 안전작업허가서 첨부·현장 비치 용도의 1~2장 요약 양식(작업단계·요인·대책·현장확인 체크란 포함)으로 출력할 수 있다(FR-ASMT-009 확장). |

### B.7 산업안전보건위원회 보고 (REC 확장)

> 근거: 절차서 5.1.2(안전관리관 역할: 산업안전보건위원회 보고).

### C.6 §5.4.4 CHARM 시트 보강 (산정 규칙 정밀화)

기존 FR-ASMT-C-002의 "자동 제안"에 다음 절차서 규칙을 디폴트 산정 로직으로 명시한다.

**노출수준(가능성) 산정**

1. 작업환경측정 결과가 있는 경우(별표5.4): 노출수준(%) = 측정결과/노출기준 × 100 → **100% 초과=4 / 50~100%=3 / 10~50%=2 / 10% 이하=1**.
2. 측정 결과가 없는 경우(별표5.5): 하루 취급량(3=ton·㎥ / 2=kg·ℓ / 1=g·㎖) × 비산성(분진·흄: 고3/중2/저1) 또는 휘발성(가스·증기: 끓는점 기준 — 상온 시 끓는점<50℃=고, 50~150℃=중, >150℃=저; 공정온도 X℃ 시 <2X+10=고, 2X+10~5X+50=중, >5X+50=저) 조합표로 1차 산정 후, **밀폐·환기상태 보정을 차감**한다(원격조작·완전밀폐=−2, 국소배기장치 설치=−1).
3. **직업병 유소견자가 발생한 물질은 측정결과·취급량과 무관하게 노출수준 = 4 강제**.

**유해성(중대성) 산정**

1. 노출기준 설정 물질(별표5.6): 분진 0.01㎎/㎥ 이하=4 / 0.01~0.1=3 / 0.1~1=2 / 1~10=1, 증기 0.5ppm 이하=4 / 0.5~5=3 / 5~50=2 / 50~500=1.
2. 노출기준 미설정 또는 초과(분진 10㎎/㎥·증기 500ppm 초과) 물질(별표5.7): MSDS의 **H-code(유해·위험문구)/R-phrase 매핑표**로 결정(예: H350·H340·H330=4, H331·H314·H317=3, H332·H302=2, H319·H315=1, 미분류 기타=1). 매핑표를 코드 마스터로 탑재한다.
3. **CMR(발암성·생식세포 변이원성·생식독성) 1A/1B/2 물질은 노출기준과 무관하게 유해성 = 4** (기존 PRD는 "1A/1B 등"으로 표기 — 절차서 기준으로 **2 구분 포함**을 명확화).

**공통**

- **혼합물질은 구성 단일물질별 위험성 계산 결과 중 최고값을 그 혼합물질의 위험성으로 결정**한다.
- 행에 산정 근거 스냅숏(유소견자 여부, 측정결과, 하루 취급량, 비산/휘발성, 밀폐·환기 보정, CMR 여부, 노출기준, H-code)을 저장·표시한다(별표5.8 실시표의 근거 열 대응).
- 화학물질 마스터(FR-ASMT-C-001) 필드 보강: 제조/사용 구분, 사용 용도, **월 취급량, 직업병 유소견자 발생 여부**, MSDS 보유 여부(별표5.1 화학물질목록). MSDS 참조 위치 안내(노출기준→§8, 물성→§9, CMR→§11, H-code→§2)를 도움말로 제공.

## F. 절차서와 기존 PRD의 충돌 사항 — 처리 결정 (PRD 우선)

병합 시 기존 PRD 본문은 수정하지 않고, 아래 차이를 비고·오픈 이슈로 관리한다.

### F.1 등급별 조치기한

| 등급(점수) | 기존 PRD §5.2.1 | 절차서 별표1.1 <표4> | 처리 |
|---|---|---|---|
| 매우 위험(13~16) | 즉시 | 즉시 개선조치(**즉시 작업중지 포함**) | PRD 유지. 단 "작업중지 검토" 경고 문구를 매우 위험 등급 안내에 추가 권고 |
| 위험(10~12) | 빠른 시일 내 | 신속하게(**1~2개월 이내**) | PRD 유지. 절차서 일수를 FR-CFG-004 디폴트 기한(D-day) 후보로 채택 → **오픈 이슈 7 해소 근거** |
| 다소 위험(7~9) | 1개월 내 | 가급적 빨리(**3~6개월 이내**) | PRD의 "1개월"이 절차서보다 엄격 — **PRD(엄격한 쪽) 유지**, 사업장 설정으로 완화 가능 |
| 위험성 존재(5~6) | 조치 필요 | 계획적으로(**정기보수공사 또는 1년 이내**) | PRD 유지. "정기보수 시 조치" 옵션을 조치 기한 유형에 추가 검토 |
| 허용 가능(1~4) | 불필요 | 필요에 따라(정보제공, 비용대비 효과 고려) | PRD 유지. "권고 조치(선택)" 등록은 허용됨을 명시 |
