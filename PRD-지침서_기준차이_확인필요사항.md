# PRD–지침서 기준 차이 확인 필요사항

| 항목 | 내용 |
|---|---|
| 목적 | PRD와 지침서(절차서)의 기준이 달라 **실무 확인·의사결정이 필요한 항목만** 발췌 |
| 기준 문서 | PRD v2.2 (`페르소나 지침서기준 수정버전.md`) ↔ `[KDHC(  )-SHP-04] 안전보건 위험성평가 절차서 R7`(2025.9) ↔ 고용노동부 고시 제2024-76호 등 |
| 처리 원칙 | **실제 업무 기준은 지침서**. 단, 고시(법령)는 지침서보다 상위 규범이므로 지침서에 없는 법정 제도는 "불필요"가 아니라 **실무 운영 여부를 확인**해야 한다 |
| 작성일 | 2026-07-08 |
