## 기준 차이 및 최종 결정 — 4단 비교표

| # | 항목 | ① PRD 기준 (v2.2) | ② 지침서 기준 | ③ 기타 법령 기준 | ④ 최종 결정 (2026-07-08) |
|---|---|---|---|---|---|
| 1 | **감소대책 필수 기준선** | **5점 이상** 감소대책 필수, 미입력 시 결재 차단 (FR-ASMT-003). 기준선은 설정형(FR-CFG-002) | **내부 불일치**: 별표1.1 <표4>는 Ⅱ(5~6)부터 허용 불가능(= **5점**), 별표1.2 순서도 주석은 "위험성**6**↑", 별표1.6 예시는 "5 이상" | 고시 제12조: 허용 불가능한 위험성에 감소대책 수립·실행 의무 (구체 점수 기준 없음) | ✅ **5점으로 확정.** <표4>·별표1.6과 일치. PRD 현행 유지, 추가 수정 없음 |
| 2 | **평가 기법 범위** | HAZOP·KRAS·JSA·CHARM **4개 기법만** 구현 (§1.4, §11) | 6.1: 수시용 **체크리스트법**, 공정용 **K-PSR**(정성)·**CA**(정량)도 지정 기법에 포함 | 고시 제7조: 기법 선택은 사업장 자율 | ⏸ **추후 선택.** 현 단계는 4개 기법 유지, 체크리스트법·K-PSR·CA 편입은 Phase 3 이후 별도 결정 (PRD 오픈 이슈 #11 유지) |
| 3 | **상시평가 (월·주 사이클)** | 「정기+수시」/「상시평가」 모드 선택형, 월·주 사이클 관리 (FR-SCHD-001, 005~007 — Phase 2) | **규정 없음** — 정기(8장) + 도급 시 수시(9장) 체계만 존재 | 고시 제15조: 상시평가(월·주·일) 이행 시 수시+정기평가 갈음 가능 (선택 제도) | ✅ **PRD 기준 채택.** 상시평가 모드 기능 그대로 구현(Phase 2). 지침서에 없어도 법정 선택 제도로서 시스템이 지원 |
| 4 | **자사 내부(비도급) 수시평가** | 고시 6대 사유 트리거 등록 + 작업 개시 전 미승인 **경보** (FR-SCHD-003/004) | 수시평가 실시 체계는 **9장 도급 시만 규정** — 자사 자체 작업의 수시 절차·결재선 조항 없음 | 고시 제15조: 6대 사유 발생 시 작업 개시 전 실시 의무 | ✅ **기능은 유지하되 제한은 미적용.** 자사 내부 수시평가의 작성·트리거 등록은 그대로 지원하고, 개시일 미승인 경보 등 강제 장치는 걸지 않음(도급 수시에는 기존대로 적용). → PRD FR-SCHD-004에 비도급 예외 반영 필요 |
| 5 | **최초평가 기한 기산점** | 신규 사업장 등록 시 **개설일 기준 1개월** 카운트다운 (FR-SCHD-008) | 8.1.1: 건축물·열원설비 등 **준공 후** 실시. 도급 업무는 착수(공) 시 제출받은 수급업체 평가로 **갈음 가능**(계획 수립 생략) | 고시 제15조: 사업 성립(개설) 후 1개월 이내 착수 | ✅ **절충 채택.** PRD대로 개설일 기준 카운트다운은 유지하되, **지침서 조건 충족 시 완료 처리**: ① 준공 후 실시한 최초평가서 연결 시 완료 ② 도급 업무는 수급업체 제출 평가 연결 시 '최초평가 갈음'으로 완료. → PRD FR-SCHD-008에 완료 조건 반영 필요 |
| 6 | **Ⅱ등급(5~6) 조치기한 환산** | 절차서 기한을 FR-CFG-004 D-day 디폴트로 채택 — 단 정기보수 일정 데이터 없음 | 별표1.1 <표4>: 계획적으로 개선조치 (**정기보수공사 또는 1년 이내**) | 고시: 구체 기한 규정 없음 | ✅ **지침서 채택.** 조치 기한 유형에 "정기보수공사 시 조치" 옵션 제공(차기 보수 예정일 직접 입력), 미입력 시 **1년(365일) 디폴트**. → PRD FR-CFG-004에 기한 유형 반영 필요 |
| 7 | **문서 유형별 결재선** | 작성→검토→승인 3단계 기본 + 결재선 템플릿 설정형 (FR-APRV-001, FR-CFG-006), Ⅴ등급 포함 시 승인권자 상향 (FR-APRV-006) | 평가서·감소대책·조치결과 = **부장 확인·승인(전결)** / 실시계획·결과 보고 = 안전관리관 작성 → **지사장 승인** / 협력사 평가 = 차·팀장 검토 → 부장 확인 (5장, 8장, 9.2) | 규정 없음 — 사업장 자율 | ✅ **지침서 채택.** 결재선 템플릿 디폴트를 지침서 계층으로 확정: 평가서는 부장 전결(지사장 단계 없음), 실시계획·결과 보고만 지사장 승인, Ⅴ등급 상향 승인 대상 = 지사장. → PRD FR-CFG-006 디폴트 값으로 반영 필요 |
| 8 | **수급업체 공유·정보 제공 수단** | 시스템 내 공유 발행·열람 서명 (FR-SHRE-001~004), 안전보건정보 패키지 발행·열람 확인 (FR-SHRE-008) | 8.4.2: 공유는 "**공문, 업무협조문 등**" / 9.1.1: 정보 제공은 "**전자계약시스템 등**" | 산안법 제64·65조: 도급인의 정보 제공·확인 의무(수단 불특정) / 고시 제13조: 결과 게시·주지 의무 | ✅ **PRD 채택.** 시스템 발행·열람 확인 기록을 공유·정보제공의 기본 수단으로 운영(지침서 문구의 "등"에 포함되는 것으로 해석). 공문 병행이 필요한 경우는 운영에서 처리. PRD 현행 유지, 오픈 이슈 #9 종결 |
| 9 | **등급 명칭·서식** | 지침서 명칭(Ⅴ~Ⅰ)을 디폴트로 채택, 구 PRD 명칭(매우위험~허용가능)은 설정 옵션 (§5.2.1~5.2.2) | 별표1.1 <표4>: **Ⅴ 매우높음 / Ⅳ 높음 / Ⅲ 약간높음 / Ⅱ 보통 / Ⅰ 낮음** | 규정 없음 — 사업장 자율 | ✅ **지침서 채택 확정.** Ⅴ~Ⅰ 로마자+명칭을 화면·출력물 표준 라벨로 확정(PRD v2.2 현행과 동일). 구 명칭 설정 옵션은 유지하되 사용하지 않음 |

## 결정에 따른 PRD 후속 반영 필요 사항

결정 자체로 PRD 수정이 필요한 항목은 3건이다 (나머지는 PRD v2.2 현행과 일치).

| 결정 # | 반영 대상 | 반영 내용 |
|---|---|---|
| 4 | FR-SCHD-004 | 작업 개시일 미승인 경보를 **도급(협력사 참여) 수시평가에만 적용**하고, 자사 내부(비도급) 수시평가는 경보 없이 작성·연결만 지원하는 예외 추가 |
| 5 | FR-SCHD-008 | 최초평가 카운트다운의 **완료 조건** 추가: ① 최초평가서 연결(준공 후 실시) 시 완료 ② 도급 업무는 수급업체 제출 평가 연결 시 '최초평가 갈음' 완료 처리 |
| 6, 7 | FR-CFG-004, FR-CFG-006 | Ⅱ등급 기한 유형에 "정기보수공사 시 조치(예정일 입력, 미입력 시 1년)" 추가 / 결재선 템플릿 디폴트를 지침서 계층(평가서=부장 전결, 계획·보고=지사장, Ⅴ등급 상향=지사장)으로 확정 |

그 외: 결정 1·3·9는 PRD v2.2 현행 유지, 결정 2는 오픈 이슈 #11 유지(추후 선택), 결정 8은 오픈 이슈 #9 종결 처리.
