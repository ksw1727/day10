# PRD–지침서 기준 차이 확인 필요사항

| 항목 | 내용 |
|---|---|
| 목적 | PRD와 지침서(절차서)의 기준이 달라 **실무 확인·의사결정이 필요한 항목만** 발췌 |
| 기준 문서 | PRD v2.2 (`페르소나 지침서기준 수정버전.md`) ↔ `[KDHC(  )-SHP-04] 안전보건 위험성평가 절차서 R7`(전면개정, 2025.9) |
| 처리 원칙 | **실제 업무 기준은 지침서**(PRD §5.2.3 절차서 우선 원칙, v2.2 반영 완료). 본 문서는 ① 지침서 내부가 불일치하거나 ② 지침서에 규정이 없거나 ③ 운영 방식 확정이 필요해 **지침서만으로 결론이 나지 않는 항목**을 다룬다 |
| 유의사항 | 고용노동부 고시(제2024-76호)는 지침서보다 상위 규범이다. **지침서에 없다 = 불필요가 아니라**, 법정 제도(상시평가 등)의 실무 운영 여부를 확인해야 하는 사안이다 |
| 작성일 | 2026-07-08 |

## 1. 감소대책 필수 기준선 — 지침서 내부 불일치 (5점 vs 6점)

| 출처 | 기준 |
|---|---|
| 지침서 별표1.1 **<표4>** (위험성 결정 기준표) | Ⅱ 보통(5~6)부터 **허용 불가능** → 5점 이상 감소조치 대상 |
| 지침서 별표1.2 순서도 주석 | "(위험성**6↑**)한 위험성에 대한 위험성 감소조치 이행" |
| 지침서 별표1.6 예시 | "5 이상" |
| PRD v2.2 | <표4> 기준 **5점 이상** 채택 (판정 기준표 우선 원칙, §5.2.3 ③) |

- **확인 요청**: 실무에서 감소조치 의무 기준선을 5점으로 운영하는지 6점으로 운영하는지. (5점이면 Ⅱ 보통 5~6점도 조치 대상, 6점이면 5점은 제외)
- **시스템 영향**: FR-ASMT-003(감소대책 미입력 시 결재 상신 차단), FR-CFG-004(자동 판정), FR-NOTI-004(고위험 자동 알림)의 기준값. 기준선은 FR-CFG-002 설정형이므로 확인 후 값만 변경하면 된다.
- **PRD 오픈 이슈**: #12

## 2. 평가 기법 범위 — 체크리스트법·K-PSR·CA 편입 여부

| 출처 | 기준 |
|---|---|
| 지침서 6.1 | 수시: (작업) JSA, **체크리스트법**, (물질) CHARM / 공정: 정성적(HAZOP, **K-PSR**), 정량적(**CA**), CHARM, JSA |
| PRD v2.2 | HAZOP·KRAS·JSA·CHARM 4개 기법만 구현(§1.4). 체크리스트법·K-PSR·CA는 범위 제외(§11) |

- **확인 요청**: 3개 기법을 시스템에 편입할지, 편입한다면 시기(Phase 3 이후 확장 검토로 유예 가능한지). §5.2.3 절차서 우선 원칙에서 **유일하게 절차서를 따르지 않은 항목**이며, 판정기준이 아닌 제품 범위 결정 사항으로 분류해 보류한 상태다.
- **참고**: 체크리스트법은 지침서 별표4에 실시표 양식·재해 재발방지용 신속 평가 용도가 규정되어 있어 실무 수요 가능성이 있다.
- **PRD 오픈 이슈**: #11

## 3. 상시평가 모드(월·주 사이클) — 지침서에 규정 없음

| 출처 | 기준 |
|---|---|
| 고시 제15조 (법규) | 상시평가(월·주·일) 이행 시 수시+정기평가 갈음 가능 |
| 지침서 | **상시평가 규정 없음** — 정기(8장) + 도급 시 수시(9장) 체계만 규정 |
| PRD v2.2 | 「정기+수시」 모드와 「상시평가」 모드 선택형(FR-SCHD-001), 월·주 사이클 관리(FR-SCHD-005~007, Phase 2) |

- **확인 요청**: 상시평가 모드를 실제로 운영할 계획이 있는지. 지침서 체계(정기+수시)만 쓴다면 상시평가 모드는 Phase 2 범위에서 후순위로 조정 가능하다. 반대로 향후 상시평가 전환 계획이 있다면 지침서 개정이 선행되어야 한다.
- **연관 이슈**: 일 단위 주지(TBM)의 갈음 요건 해석(기존 오픈 이슈 #4)도 이 결정에 종속된다.

## 4. 자사 내부(비도급) 수시평가 — 지침서에 절차 없음

| 출처 | 기준 |
|---|---|
| 고시 제15조 (법규) | 수시평가 사유 6종: ① 건설물 설치·이전·변경·해체 ② 기계·기구·설비·원재료 신규 도입/변경 ③ 정비·보수 ④ 작업방법·절차 신규 도입/변경 ⑤ 산업재해·아차사고 ⑥ 중대산업사고 |
| 지침서 | 수시평가 실시 체계는 **9장 "도급시 수시 위험성평가"만 규정** — 자사 자체 작업의 수시평가 절차(승인 라인·서식)는 별도 조항 없음 |
| PRD v2.2 | 고시 6대 사유의 트리거 이벤트 등록·작업 개시 전 승인 경보(FR-SCHD-003/004) |

- **확인 요청**: 도급이 아닌 자사 작업(설비 변경, 정비·보수 등)의 수시평가를 실무에서 어떤 절차(작성·검토·승인 주체, 사용 서식)로 운영하는지. 정기평가 체계(차·팀장 작성 → 부장 승인)를 준용하는지 확인 필요.
- **시스템 영향**: 수시평가서의 결재선 디폴트(FR-CFG-006), 트리거 사유 6종 목록의 유지 여부.

## 5. 최초평가 기한 기산점 — 성립 후 1개월 vs 준공 후

| 출처 | 기준 |
|---|---|
| 고시 제15조 (법규) | 사업 성립(개설) 후 **1개월 이내 착수** |
| 지침서 8.1.1 | 건축물·열원설비 등 **준공 후** 실시. 계약(도급) 업무의 최초평가는 **착수(공) 시 제출받은 수급업체 평가로 관리 가능**(이 경우 계획 수립 생략) |
| PRD v2.2 | 신규 사업장 등록 시 **개설일 기준 1개월 카운트다운**(FR-SCHD-008) |

- **확인 요청**: ① 최초평가 카운트다운의 기산일을 개설일로 할지 준공일로 할지(사업장 속성에 준공일 필드 필요 여부). ② 도급 업무의 "수급업체 제출 평가로 최초평가 갈음" 처리를 시스템에서 어떻게 기록할지(협력사 제출 평가서에 '최초평가 갈음' 표시 등).

## 6. Ⅱ등급 조치기한 "정기보수공사 또는 1년 이내" — D-day 환산 방법

| 출처 | 기준 |
|---|---|
| 지침서 별표1.1 <표4> | Ⅱ 보통(5~6): 계획적으로 개선조치 (**정기보수공사 또는 1년 이내**) |
| PRD v2.2 | 절차서 기한을 FR-CFG-004 디폴트로 채택 완료. 단 시스템에는 정기보수 일정 데이터가 없음 |

- **확인 요청**: "정기보수공사 시 조치" 선택 시 기한을 어떻게 관리할지 — ① 차기 정기보수 예정일을 수기 입력 ② 일괄 1년(365일)로 환산 ③ 정기보수 일정 마스터 별도 관리(범위 확장). 실무에서 정기보수 일정을 어디서 관리하는지에 따라 결정.

## 7. 문서 유형별 결재선 매핑 — 부장 승인 vs 지사장 승인

| 출처 | 기준 |
|---|---|
| 지침서 5장·8장 | **평가서·감소대책·조치결과**: 관리감독자(부장) 확인·승인까지 / **실시계획·결과 보고**: 안전관리관 작성 → 안전보건관리책임자(지사장) 승인 / 협력사 평가: 차·팀장 검토 → 부장 확인 |
| PRD v2.2 | §3.2에 계층 반영 완료. 전자결재는 작성→검토→승인 3단계 기본(FR-APRV-001) + 결재선 템플릿 설정형(FR-CFG-006) |

- **확인 요청**: ① 평가서 결재에 지사장 단계를 넣지 않는 것이 실무와 일치하는지(지침서상 평가서 자체는 부장 전결로 해석됨). ② 'Ⅴ 매우 높음' 포함 평가서의 승인권자 상향(FR-APRV-006) 시 상향 대상이 지사장인지, 별도 규정이 있는지. ③ 검토 단계(차·팀장) 생략 가능 문서 유형.
- **시스템 영향**: FR-CFG-006 결재선 템플릿의 문서 유형별 디폴트 값.

## 8. 수급업체 공유·안전보건정보 제공 — 시스템 갈음 여부

| 출처 | 기준 |
|---|---|
| 지침서 8.4.2 | 수급업체 공유는 "**공문, 업무협조문 등**을 통해" |
| 지침서 9.1.1 | 안전보건정보 제공은 "**전자계약시스템 등**을 통해" |

- **확인 요청**: 시스템 내 발행·열람 확인 기록이 지침서의 공문·전자계약시스템 절차를 **갈음할 수 있는지**, 아니면 병행해야 하는지(병행 시 공문 발송 이력의 수기 등록 필드 필요). 지침서 문구가 "등"으로 열려 있어 해석 확인이 필요하다.
- **PRD 오픈 이슈**: #9 (기존 이슈와 동일 사안)

## (참고) 이미 지침서 기준으로 반영 완료 — 확인 불필요

아래 항목은 v2.1~v2.2에서 지침서 기준으로 교체 완료되어 별도 확인이 필요 없다.

- 페르소나·역할 체계: 지사장/안전관리관/관리감독자(부장·차·팀장)/근로자/수급업체 (§3, 지침서 5장)
- 등급별 조치기한: 즉시(작업중지 포함) / 1~2개월 / 3~6개월 / 정기보수 또는 1년 / 필요에 따라 (§5.2.1, <표4>) — 단 Ⅱ등급 환산 방법은 위 6번
- 등급 명칭 디폴트: Ⅴ~Ⅰ 로마자+명칭 (§5.2.1~5.2.2) — 단 실운영본 대조는 위 9번
- 대책 유형 라벨: 본질적/공학적/관리적/보호구 (FR-ASMT-004)
- JSA 실시 유형: 최초·정기·수시 공용 (지침서 6.1)
- 협력사 평가 처리: P5 제출 → 차·팀장 검토(이행점검 체크리스트) → 부장 확인 (지침서 9.2)
- 조치 확인권자: 관리감독자(부장) (FR-ACT-004, 지침서 8.5.3)
- 실시계획 결재: 안전관리관 작성 → 지사장 승인 (FR-SCHD-010, 지침서 8.1.1)
- 타 기법 적용 절차: 지침 작성(안전관리관)·승인(지사장) 후 적용 (지침서 6.2)
- 위험도 축 정의문·중대성 재산손실 병기, CHARM 산정 규칙, 공정분류 마스터 등 서식·판정 규칙 (§5.2.2, §5.4 각 보강 사항)
